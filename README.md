# PONG_IOTA
Micropayments PoC

### Requirements
```
pip install qrcode
pip install pygame
pip install pyota[ccurl]
```
### How to start
```
git clone https://github.com/MichiKuhl/PONG_IOTA.git
cd pong_iota_v1
python pong.py
```
